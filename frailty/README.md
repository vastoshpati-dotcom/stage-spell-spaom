# Process-Frailty Sensitivity Analysis

This folder contains the process-level random-intercept sensitivity analysis
for the Stage–Spell Peace Agreement Opportunity Model (SS-SPAOM).

The frailty specification examines whether the estimated stage-spell effect
remains after accounting for persistent unobserved heterogeneity among peace
processes.

The fixed-effects SS-SPAOM remains the primary model; the frailty model is
reported as a sensitivity analysis.
